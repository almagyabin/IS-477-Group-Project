# Data Dictionary

- mal_id — Unique anime ID (MyAnimeList)
- title — Anime title
- score_csv — User rating score (1–10)
- favorites — Number of users marking the anime as favorite
- year — Release year
- genres — List of genres
- studios — Animation studio
- rating_csv — Age rating
- duration — Episode duration
- aired_from / aired_to — Broadcast dates
- anime_id — Unique identifier from JSON dataset (same as mal_id)
